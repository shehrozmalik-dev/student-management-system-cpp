# student-management-system-cpp

## Description
OOP based student management system in c++ using arrays.

## features
1. Add Student
2. Add Instructor
3. Add Course
4. Enroll Student in Course
5. Assign Instructor to Course
6. Show All Students and Instructors
7. Assign Student Grade
8. Assign New Instructor to Course
9. Show All Students
10. Show All Instructors
11. Delete Student
12. Delete Instructor
13. View Course
14. View All Courses
15. Delete Course
16. Search Student
17. Search Instructor
18. Export Data To Files

## Concepts used
1. classes and objects
2. Encapsulation
3. Inheritance
4. File handling
5. Abstraction
6. Composition
7. Polymorphism
